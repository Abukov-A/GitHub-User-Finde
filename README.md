# GitHub User Finder

## Автор
Абуков Амир

## Описание
Приложение для поиска пользователей GitHub с возможностью добавления в избранное и сохранения списка в JSON.

## Использование API
Для поиска используется GitHub API: GET https://api.github.com/users/{username}

## Примеры использования
1. Введите логин пользователя в поле поиска и нажмите «Поиск».
2. В результатах выберите пользователя и нажмите «Добавить в избранное».
3. Избранные пользователи сохраняются в файл favorites.json.

Для работы вам понадобятся библиотеки aiohttp и Pillow. Установите их командой:pip install aiohttp Pillow

Как использовать
Сохраните этот код в файл с именем main.py.
Установите необходимые библиотеки: pip install aiohttp Pillow.
Запустите программу: python main.py.
Введите логин пользователя GitHub (например, gigachat) и нажмите «Поиск».
Данные отобразятся в таблице. Нажмите «⭐ Добавить в избранное», чтобы сохранить пользователя.
Список избранных будет сохранен в файл favorites.json в той же папке.
